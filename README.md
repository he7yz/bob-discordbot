A personal-made discord.js bot tailor-made for MMUCraft Discord

# Requirements
Install `npm` & `node.js`(v22.19.0 LTS)
```bash
$ npm install
```
Then, install `discord.js` (v14.22.1)
```bash
$ npm install discord.js
```
Next, install `dotenv`
```bash
$ npm install dotenv
```
Use `.env_example` and rename it to `.env`, then insert the token within.

To start, run
```
$ npm .
```
